두 양의 정수 a, b (a > b)에 대하여, 
a와 b의 최대공약수는 b와 a를 b로 나눈 나머지(r)의 최대공약수와 같다.
즉, GCD(a, b) = GCD(b, a % b)

예시: 1071과 1029의 최대공약수 구하기
1071을 1029로 나눈 나머지는 42입니다.
GCD(1071, 1029) = GCD(1029, 42)
1029를 42로 나눈 나머지는 21입니다.
GCD(1029, 42) = GCD(42, 21)
42를 21로 나눈 나머지는 0입니다.
GCD(42, 21) = 21

#include <stdio.h>

// 유클리드 호제법으로 최대공약수(GCD)를 구하는 함수
int getGCD(int num1, int num2) {
    int temp;

    // num2가 0이 될 때까지 반복
    while (num2 != 0) {
        temp = num1 % num2; // num1을 num2로 나눈 나머지를 temp에 저장
        num1 = num2;        // num2를 num1에 저장
        num2 = temp;        // temp(나머지)를 num2에 저장
    }
    // 반복이 끝나면 num1에 최대공약수가 남음
    return num1;
}

int main(void) {
    int a, b;

    printf("두 개의 정수를 입력하세요: ");
    scanf("%d %d", &a, &b);

    int gcd = getGCD(a, b);

    printf("두 수의 최대공약수는 %d입니다.\n", gcd);

    return 0;
}
